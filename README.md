# File Converter API

A file conversion REST API built with FastAPI and libraries like LibreOffice headless.
This API allows users to upload files (Word, Excel, PowerPoint, images, etc.) and convert them into different formats such as PDF, DOCX, XLSX, CSV, TXT, and more.

---

## Features

* Upload and convert files
* Supports multiple formats
* FastAPI REST API
* LibreOffice headless conversion engine
* Docker support
* Scalable architecture
* Ready for cloud deployment

---

## Supported Conversions

| From    | To  |
| ------- | --- |
| DOCX    | PDF |
| XLSX    | PDF |
| PPTX    | PDF |
| ODT     | PDF |
| TXT     | PDF |
| JPG/PNG | PDF |
| XLSX    | CSV |
| DOCX    | TXT |
